## Install / Run (Local)

### Prerequisites
- Node.js + npm

### Install dependencies
From the repo root:
- `npm install`
- `npm --prefix client install`
- `npm --prefix server install`

### Run the app (client + server)
From the repo root:
- `npm run dev`

### Storybook (UI component lab)
From the repo root:
- Run Storybook: `npm --prefix client run storybook`
- Build Storybook: `npm --prefix client run build-storybook`
